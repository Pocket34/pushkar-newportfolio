# Pushkar Gupta Portfolio

A futuristic personal portfolio website built using React, TypeScript and Tailwind CSS.

## Live Demo

https://pushkar-newportfolio.vercel.app

## Features

* Responsive Design
* Cyberpunk UI
* Projects Showcase
* Resume Download
* Contact Section
* Social Links Integration

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite
* Vercel
